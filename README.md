## About This Repository

This repository serves as a project template designed for students enrolled in the Software Testing course at Tampere University.

The repository includes only the implementation code being tested, along with the `package.json` configuration file and LICENSE documentation.

## Getting Started

This project has been verified to function properly with Node.js version 16.0.0

### Installation

To install the required dependencies:

```bash
npm install
```

### Execute Tests

To run the test suite:

```bash
npm test
```

### Generate Coverage Report

To run tests and generate a comprehensive test coverage report:

```bash
npm run test-coverage
```
